# emotion_journal

日常の感情と身体状態を記録し、  
**感情 × 身体エネルギー**を可視化するための個人用ジャーナルアプリ。

---

## Purpose（目的）

- その時々の心身状態を、感覚ではなく**構造として把握**する  
- 感情の偏りやエネルギー低下を**早期に視覚認識**する  
- 日々の状態を履歴として蓄積し、**振り返りと自己調整**に活かす

---

## Success Criteria（成功条件）

- 今の心身状態が**一目で分かる**
- 感情と身体エネルギーの**傾向が視覚で理解できる**
- 記録が履歴として残り、**時系列で確認できる**

---

## Concept（基本思想）

- 思考は **感情 × 身体エネルギー** の関係性として捉える  
- 可視化は **2次元平面（感情軸 × エネルギー軸）** で行う  
- グループ分類により **現在の心身状態の所属領域**を把握する

- GitHub（仕様・コード・履歴管理）
- manus（設計・思考・試行錯誤の履歴）

---

## Development Policy（開発方針）

- **manus**  
  思考・設計・試行錯誤の履歴を残す場所
- **GitHub**  
  仕様とコードの正史（確定情報）
- **GAS**  
  実行処理と自動化の中核
